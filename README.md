# wol — Wake-on-LAN утилита

CLI утилита для включения компьютера по сети через магический пакет.

## Использование

```bash
wol D8-BB-C1-72-DA-74                        # включить по MAC (Windows формат)
wol D8:BB:C1:72:DA:74                        # включить по MAC (Linux формат)
wol D8-BB-C1-72-DA-74 -b 10.192.128.255     # указать broadcast адрес
wol D8-BB-C1-72-DA-74 -p 7                  # указать порт
```

## Флаги

| Флаг | Короткий | По умолчанию | Описание |
|------|----------|--------------|----------|
| `--broadcast` | `-b` | `255.255.255.255` | Broadcast адрес сети |
| `--port` | `-p` | `9` | UDP порт |

## Как узнать MAC адрес

**Windows:**
```cmd
ipconfig /all
# ищи строку "Физический адрес"
```

**Linux/macOS:**
```bash
ip link show
# или
ifconfig
```

## Как работает

Отправляет "магический пакет" по UDP broadcast — 6 байт `0xFF` и MAC адрес повторённый 16 раз. Сетевая карта компьютера получает пакет даже в выключенном состоянии и подаёт питание на материнскую плату.

## Требования

- Go 1.21+
- Wake-on-LAN должен быть включён в BIOS целевого компьютера
- Оба устройства должны быть в одной локальной сети

## Установка

```bash
git clone https://github.com/hitprim/wol
cd wol
go build -o wol.exe .\cmd\wol\
```
